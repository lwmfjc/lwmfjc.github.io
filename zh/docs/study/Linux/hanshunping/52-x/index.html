    磁盘分区 #

    • 分区跟文件系统的关系 (挂载)
      • 将一个分区挂载到某个目录，用户进入到某个目录，就相当于访问到某个分区了
    • lsblk
    • linux分IDE硬盘和SCSI硬盘 目前基本是SCSI硬盘
    • sdx~ x代表abcd，~表示数字
    • lsblk -f
      • 文件类型，唯一标识符
    • 现在挂载一个分区
      • 如图
      • 给虚拟机，添加一个硬盘
      • 重启后，使用lsblk
      • 进行分区
        • fdisk /dev/sdb
        • 之后输入p，
        • 输入分区数（这里是1）
        • 最后一步，输入w ，写入分区并退出
        • 查看
      • 将分区格式化 mkfs -t ext4 /dev/sdb1
        • 查看
      • 进行挂载 mount /dev/sdb1 /newdisk/
      • umount /dev/sdb1 卸载

      • 用命令行挂载的指令，重启后挂载关系会消失 永久挂载：修改/etc/fstab #

    • df -h 查看磁盘使用情况
    • du -h –max-depth=1 /opt

    • ls -lR /opt 注意，这里加了R，将递归显示
