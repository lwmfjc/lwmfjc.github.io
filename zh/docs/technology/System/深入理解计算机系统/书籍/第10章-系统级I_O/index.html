    Linux 提供了少械的基千 U nix I/ 0 模型的系统级函数．它们允许应用程序打开、关闭、读和写文件， 提取文件的元数据 ， 以及执行 I/ 0 重定向。Linux 的 读和写操 作会出现不足值， 应用程序必须能正确地 预计和处理这种情 况。应用 程序不 应直接调用 Unix I/ 0 函数， 而应该使用 RIO 包， RIO 包通过反复执行读写操作，直到传送完所有的请求数据，自动处理不足值。
